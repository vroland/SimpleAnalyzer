vector< float > *  out,
vector< float > *  times,
vector< int > *  debug_positions,
size_t  row_count 
)

Sammelt Daten aus einer Textzeile (string).

Parameter
lineDie zu untersuchende Textzeile.
outAusgabevariable für die Sensordaten der Zeile. Alle Spalten nach opts.start_col werden als Sensordatenspalten betrachtet. Wenn row_count == opts.startrow ist, werden statt der Sensordaten die Faserpositionen eingelesen!
timesWenn nicht NULL, Ausgabevariable für den Zeitstempel der Zeile (opts.timecol). Der Zeitstempel wird an die übergebene Liste angehängt.
debug_positionsWenn nicht NULL, Ausgabevariable für die Position der einzelnen Messwerte in der Datei. Diese Positionen werden in der Logdatei zur Fehlerkorrektur angegeben, um ein Wiederfinden der Werte für den Nutzer einfacher zu machen.
row_countNummer der aktuellen Zeile (Index+1).
